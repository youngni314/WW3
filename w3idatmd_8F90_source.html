87  ! ICEI R.A. Public Ice concentrations.
88  ! UX0/N R.A. Public Cartesian X and Y momentum components
89  ! UY0/N R.A. Public for both times.
90  ! RH0/N R.A. Public Air density for both times
91  ! BERGI R.A. Public Iceberg damping coefficient
92  ! IINIT Log. Public Flag for array initialization.
93  ! FLLEV Log. Public Flag for water level input.
94  ! FLCUR Log. Public Flag for current input.
95  ! FLWIND Log. Public Flag for wind input.
96  ! FLICE Log. Public Flag for ice input.
97  ! FLTAUA Log. Public Flag for atmospheric momentum input
98  ! FLRHOA Log. Public Flag for air density input
99  ! INFLAGS1 L.A. Public Array consolidating the above six
100  ! flags, as well as four additional
101  ! data flags.
102  ! INFLAGS2 L.A. Public Like INFLAGS1 but does *not* get changed
103  ! when model reads last record of ice.ww3
104  ! FLAGSC L.A. Public Coupling or not for input variables
105  ! JFIRST Int Public First index of arrays related to
106  ! input fields. At present this is
107  ! hardwired below. Field-related arrays
108  ! (e.g., INFLAGS1) will be allocated from
109  ! JFIRST:9 (e.g., ALLOCATE(INFLAGS1(JFIRST:9))).
110  ! CXTIDE R.A. Public Tidal constituents of X current component
111  ! CYTIDE R.A. Public Tidal constituents of Y current component
112  ! WLTIDE R.A. Public Tidal constituents of water level
113  ! FLLEVTIDE Log. Public Flag for use of tidal const. in water level input.
114  ! FLCURTIDE Log. Public Flag for use of tidal const. in current input.

360  !

395  inputs(i)%IINIT = .false.
396  inputs(i)%INFLAGS1 = .false.
397  inputs(i)%INFLAGS2 = .false.
398  inputs(i)%FLAGSC = .false.
399  END DO
400  !
401 #ifdef W3_T
402  WRITE (ndst,9000) -nauxgr, ngrids
403 #endif
404  !
405  RETURN
